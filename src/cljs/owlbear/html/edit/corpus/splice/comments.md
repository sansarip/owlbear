# HTML Splice

## Nonempty Comments
```html
▌<!-- Hello -->
```
```html
▌Hello 
```

```html
<!-- ▌Hello -->
```
```html
▌Hello 
```

```html
<!-- Hello, 
▌World!
-->
```
```html
Hello, 
▌World!

```

```html
<div>
	▌<!--<h1>Hello</h1>-->
</div>
```
```html
<div>
	▌<h1>Hello</h1>
</div>
```

## Empty Comments
```html
▌<!---->
```
```html
❎
```

```html
<!----▌>
```
```html
❎
```
