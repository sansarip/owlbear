# TypeScript Forward Move
## Spread Syntax
```typescript
[▌...foo, bar]
```
```typescript
[...foo, ▌bar]
```

```typescript
[..▌.foo, bar]
```
```typescript
[...foo, ▌bar]
```

```typescript
[...▌foo, bar]
```
```typescript
[...foo, ▌bar]
```

```typescript
[...▌foo(), bar]
```
```typescript
[...foo(), ▌bar]
```

```typescript
[...▌foo.bar(), baz()]
```
```typescript
[...foo.bar(), ▌baz()]
```
