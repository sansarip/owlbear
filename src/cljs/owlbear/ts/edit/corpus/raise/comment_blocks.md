# TypeScript Raise
## Basic Comment
```typescript
/*▌Hello, World!*/
```
```typescript
▌Hello, World!
```

## Nested Comments
```typescript
/*▌\/**\/*/
```
```typescript
▌/**/
```

```typescript
/*▌\/*\/**\/*\/*/
```
```typescript
▌/*\/**\/*/
```

```typescript
/*
▌const foo = () => {
  \/* Hello, World! *\/
  // Buh!
  return 1 + 1;
}
*/
```
```typescript

▌const foo = () => {
  /* Hello, World! */
  // Buh!
  return 1 + 1;
}

```
